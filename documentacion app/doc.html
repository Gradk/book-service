Установка и запуск проекта

Для запуск апроекта требуется актуальная версия node js.

Установка зависимостей

              
                $ npm i

Запуск проекта

              
                //запуск в dev режиме
                $ npm run start:dev

Исходные данные

исходные данные проекта находятся в корневой директории в файле .env.

User API

Управление User моделью с помощью Nest js + Mongoose

User Model Schema

            
             User {
                name: string;
                email: string;
                password: string;
             }
Поле Свойства Тип
name required: true string
email required: true, unique: true string
password required: true string

Поля будут добавляться.

Добавление Пользователя

Post

            
            /users/register/

Требуемые данные для создания юзера

            
              CreateUserDto {
                name: string;
                email: string;      
                password: string;
              }
